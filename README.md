## 📌 About This Repository

This repository contains the complete solutions and notes for **Cognizant's Digital Nurture 5.0** program — a structured, hands-on training program for freshers to develop full-stack engineering skills.

The program is split into **two phases**:

| Phase | Name | Duration | Focus |
|-------|------|----------|-------|
| Phase 1 | **Upskilling** | Pre-program | HTML, CSS, JavaScript, Bootstrap, jQuery, SQL, C# |
| Phase 2 | **Deep Skilling** | 7 Weeks | Design Patterns, EF Core, ASP.NET Core, Microservices, Angular, Docker |
