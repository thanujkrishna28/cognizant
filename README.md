# Cognizant Digital Nurture 5.0 Program

This repository contains the coursework, exercises, and projects completed during the **Cognizant Digital Nurture 5.0 Program**.

## Repository Structure

The repository is divided into two primary tracks:

### 1. 🚀 [Upskilling Track](./Upskilling/README(UpSkilling).md)
Focuses on the fundamentals of full-stack engineering.
* **[Module 1 — Frontend Fundamentals](./Upskilling/Module%201)**: HTML5, CSS3, JavaScript, Bootstrap 5, and jQuery.
* **[Module - 2 — Database](./Upskilling/Module%20-%202)**: ANSI SQL using MySQL database design and queries.
* **[Module - 3 — Backend](./Upskilling/Module%20-%203)**: C# and ADO.NET database access patterns.

### 2. 💻 [Deepskilling Track](./Deepskilling/DEEPSKILLING_README.md)
Advanced software engineering concepts, design patterns, testing, frameworks, and architecture.
* **Week 1**: Design Patterns and DSA.
* **Week 2**: Advanced SQL Server, NUnit, and Moq.
* **Weeks 3–7**: Entity Framework Core 8, ASP.NET Core Web API, Microservices, Angular, and DevOps (Docker, CI/CD).
