## 📚 Module Details

### Module 1 — Design Patterns & Principles
Covers SOLID principles and Gang of Four (GoF) patterns implemented in C#:
- **Creational**: Singleton, Factory Method
- **Structural**: Adapter, Decorator, Proxy
- **Behavioral**: Observer, Strategy, Command
- **Architectural**: MVC, Dependency Injection

### Module 2 — Data Structures & Algorithms
Focus on complexity analysis and implementation:
- Big O / Best / Average / Worst case analysis
- Sorting: Bubble, Insertion, Merge, Quick, Heap
- Searching: Linear, Binary
- Linked Lists, Arrays, Recursion

### Module 3 — Advanced SQL (SQL Server)
Enterprise-level SQL beyond basics:
- Window functions: `OVER`, `PARTITION BY`, `RANK`, `DENSE_RANK`
- CTEs, MERGE, PIVOT / UNPIVOT
- Stored Procedures, Scalar & Table-valued Functions
- Triggers (AFTER / INSTEAD OF), Cursors
- TRY/CATCH, Transactions, ACID properties, Isolation Levels

### Module 4 — NUnit & Moq
Clean unit testing practices in .NET:
- Test pyramid & TDD workflow
- Parameterized tests, Setup/Teardown
- Mocking dependencies with Moq
- State vs Interaction testing

### Module 5 — Entity Framework Core 8
ORM with code-first approach:
- DbContext, Migrations, Seed Data
- One-to-One, One-to-Many, Many-to-Many relationships
- CRUD with LINQ: `Where`, `Select`, `OrderBy`, `FirstOrDefault`
- Eager / Lazy / Explicit loading
- Performance: `AsNoTracking`, Bulk ops, RowVersion concurrency

### Module 6 — ASP.NET Core Web API
Full REST API lifecycle:
- Controllers, Action Methods, Attribute Routing
- Middleware, Filters, Global Exception Handling
- JWT Authentication & Role-based Authorization
- Swagger/OpenAPI integration
- CORS, Serilog logging

### Module 7 — Microservices Architecture
Decomposing monoliths:
- Microservices vs Monolith tradeoffs
- HTTP / Messaging / gRPC inter-service communication
- Service Discovery, Database-per-service pattern
- Rolling updates, Blue-green deployments
- CI/CD with GitHub Actions / Azure DevOps, Docker & Kubernetes basics
